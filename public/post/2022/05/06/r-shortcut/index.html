Shortcuts in R studio

Sikai Huang

2022/05/06

R shortcuts make the programming life easier. They are available directly in the Navigation Bar - Tools- Keyboard shortcuts help. I listed commonly used shortcuts that might help R beginners (students) like me:

Table: Shortcuts make life easier
Windows & Linux Mac
Open Document Ctrl + O Cmd + O
New R Script Ctrl + Shift + N Cmd + Shift + N
Run Selected Line Ctrl + Shift + ↵ Cmd + Shift + ↵
Undo Ctrl + Z Cmd + Z
Redo Ctrl + Shift + Z Cmd + Shift + Z
Copy Ctrl + C Cmd + C
Paste Ctrl + Y Cmd + Y
Cut Ctrl + X Cmd + X
Move Lines Up/Down Alt+Up/Down Option+Up/Down
Delete Line Ctrl + D Cmd + D
Move to Line start Ctrl + Left Cmd + Left
Move to Line end Ctrl + Right Cmd + Right
Select to Line Start Ctrl + Shift + Left Cmd + + Shift + Left
Select to Line End Ctrl + Shift + Right Cmd + + Shift + Right
Insert Pipe Operator( %>% ) Ctrl + Shift + M Cmd + Shift + M
Find Ctrl + F Cmd + F
