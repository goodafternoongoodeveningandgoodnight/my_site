R shortcuts aim to make the programming life easier. It is available directly in the Navigation Bar - Tools- Keyboard shortcuts help. I listed commonly used shortcuts that might help R beginners (students) like me:

  • New R script : Ctrl + Shift + N - Cmd + Shift + N
  • Run selected line : Ctrl + Shift + ↵ - Cmd + Shift + ↵
  • Undo : Ctrl + Z - Cmd + Z
  • Redo : Ctrl + Shift + Z - Cmd + Shift + Z
  • Copy : Ctrl + C - Cmd + C
  • Paste : Ctrl + Y - Cmd + Y
  • Cut : Ctrl + X - Cmd + X
  • %>% : Ctrl + Shit + M - Cmd + Shift + M
